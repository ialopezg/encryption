## Requirements

* **PHP** *v5.6+*
* **ext-openssl** PHP extension
* **ialopezg/core** *v0.0.1+*
* **ialopezg/logger** *v0.0.4+*

## Installation

1. Package manager
    - Composer
        ```shell script
        composer require ialopezg/core
        ```

2. Manually
    - **Github**
        ```shell script
        git clone https://github.com/ialopezg/core
        ```
